Upload a CSV file with attendant information. The file will be processed and attendants will be created or updated based on email addresses.
CSV Format Requirements

Your CSV file must include the following columns (case-sensitive):

  • First Name - Required
  • Last Name - Required
  • Email - Optional (used to identify existing attendants if provided)
  • Phone - Optional
  • Congregation - Optional
  • Serving As - Optional (comma-separated)
  • Preferred Positions - Optional
  • Emergency Contact - Optional
  • Emergency Phone - Optional
Serving As Values:
  • Elder
  • Ministerial Servant
  • Regular Pioneer
  • Auxiliary Pioneer
  • Publisher
  • Exemplary

Use comma-separated values for multiple positions, e.g., "Elder, Regular Pioneer"
